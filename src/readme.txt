WDXTagLib
Content plugin for Total Commander (http://www.ghisler.com/)
by Dmitrie Murzaikin


Description --------------------------------------------------------------------
	This is a content plugin for Total Commander that allows you to show audio
file tags in columns, edit tags and search in tags.


Functionality ------------------------------------------------------------------
Tags supported for reading and writing:
	• title;
	• artist;
	• album;
	• year;
	• tracknumber;
	• comment;
	• genre.

Readonly tags:
	• bitrate;
	• samplerate;
	• channels;
	• length.

It has been tested with Total Commander 7.02a under Windows XP.


Known issues -------------------------------------------------------------------
• National characters are not supported yet.

• Total Commander 7.02 and higher.


Installation -------------------------------------------------------------------
Firstly place the taglib.dll in the totalcmd.exe folder. Then add wdxtaglib.wdx
as content plugin.
